Jupyter 下载和安装

1、首先把 pip 命令升级到最新版本：

```shell
C:\Users\Administrator\Desktop
λ pip install --upgrade pip
Requirement already up-to-date: pip in c:\python375\lib\site-packages (19.3.1)
```

2、安装 Jupyter Notebook

```shell
pip install -i https://pypi.doubanio.com/simple/ jupyter
```
