# CSS 3 的模块
CSS 3 的模块可以划分如下==**十一**==部分：
1、选择器
2、文本
3、字体
4、背景
5、边框
6、框模型: 盒子模型
7、转换: 2D转换, 3D转换
8、过渡
9、动画
10、多列布局
11、用户界面
